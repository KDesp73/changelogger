# changelogger

A cli tool to help you keep a changelog in all your projects

## Installation

```bash
# Get the source code
$ git clone https://github.com/KDesp73/changelogger --depth=1

# Checkout to the latest release
$ git checkout v0.0.2

# Build and Install
$ ./install.sh
```

## Help

Try `changelogger -h` or `changelogger <command> -h`
